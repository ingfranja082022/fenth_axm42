# fenth_axm

Proyecto prototipo: página moderna para cambio de divisas.

Stack:
- React
- Vite
- TailwindCSS (Play CDN para prototipo)

Instrucciones rápidas:

1. Abrir terminal en la carpeta `fenth_axm`.
2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar en modo desarrollo:

```bash
npm run dev
```

Notas:
- La app obtiene tasas desde `https://api.exchangerate.host/latest`.
- Para un deployment/proyecto real se recomienda instalar Tailwind via PostCSS y configurar variables de entorno.
